EPIC Array CG Identification

  • Version 0.01 (03-15-2018)
  • Takes a pre-formatted .csv file and outputs a csv file with the appropriately matched CG ID numbers
  • Sample input format and sample output format.

Directions for Input

  1. Paste EpigenDX CpG #'s into column A
  2. Paste EpigenDX hg38 coordinates into column B
  3. Paste EPIC CpG ID's into column C
  4. Paste EPIC hg38 coordinates into column D
  5. Save this as a CSV file
  6. Upload the CSV file and your results will be in this format:
    • EpigenDX CpG #'s in column A
    • EPIC CpG ID's in column B
    • hg38 coordinates in column C
    • EpigenDX CpG #'s and EPIC CpG ID's in column D
    • A list of all EpigenDX CpG #'s in column E
